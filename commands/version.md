# Check CLI Version

Display the installed Firecrawl CLI version.

## Command Syntax

```bash
firecrawl version
```

## Output

Returns the version number, e.g.:
```
0.0.6
```

## When to Use

- Verifying CLI installation
- Troubleshooting issues (checking for updates)
- Confirming setup during `/firecrawl:setup`

## Related

- Use `/firecrawl:setup` to install or configure the CLI
- Check https://github.com/mendableai/firecrawl-cli for the latest version
